# User Management System 🚀

A simple **CRUD (Create, Read, Update, Delete) User Management System** built using **Node.js, Express, MongoDB, and EJS**.

## 📌 Features
- 🔹 **Create** a new user  
- 🔹 **Read** user details  
- 🔹 **Delete** a user  
- 🔹 **Styled using Tailwind CSS**  

## 🛠️ Technologies Used
- **Node.js**
- **Express.js**
- **MongoDB (Mongoose)**
- **EJS (Embedded JavaScript Templates)**
- **Tailwind CSS**

## 📦 Installation & Setup
1. **Clone the repository**
   ```sh
   git clone https://github.com/Jayanth-Rookie/user-manager.git
   cd YOUR_REPO_NAME
   ```

2. **Install dependencies**
   ```sh
   npm install
   ```

3. **Start the server**
   ```sh
   npm start
   ```
   The app runs on **http://localhost:3000**



## 🔥 Routes
| Method | Route | Description |
|--------|-------|-------------|
| GET | `/` | Home Page |
| GET | `/read` | View all users |
| POST | `/create` | Create a new user |
| GET | `/delete/:id` | Delete a user |
